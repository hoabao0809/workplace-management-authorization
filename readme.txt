# Tech Stack: 
EJS, Node.js, Express.js, MongoDB (Mongoose). 
# Function: 
The application helps employees check in/out or register absences in workplace, and also allows managers to track the absence and time reports of employees.
# Responsibility:
- Build a Node.js MVC application using Express.js, MongoDB, and use EJS template engine to implement server-side rendering.
- Use session to implement authentication and authorization
